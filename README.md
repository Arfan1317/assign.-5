## 📝 JavaScript Concept Questions

**1️⃣ What is the difference between var, let, and const?**
`var` is the older, sometimes unpredictable way to declare variables, `let` is for values you plan to change later, and `const` is for values that must stay exactly the same.

**2️⃣ What is the spread operator (...)?**
It unpacks an array or object into individual pieces, kind of like emptying a box of items onto a table so you can use them separately.

**3️⃣ What is the difference between map(), filter(), and forEach()?**
`map()` changes every item and hands you a brand new list, `filter()` hands you a new list containing only items that passed a specific test, and `forEach()` simply loops through the list to do an action without giving anything back.

**4️⃣ What is an arrow function?**
It is just a shorter, much cleaner way to write a JavaScript function without actually having to type out the word "function".

**5️⃣ What are template literals?**
They allow you to easily inject variables directly into a string of text using backticks (\`) and `${}` instead of awkwardly chaining words and variables together with plus signs.
